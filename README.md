# Эволюционный поиск 8×8 S-box

Проект реализует эволюционный поиск биективных 8×8 S-box с хорошими криптографическими свойствами (нелинейность, дифференциальная и линейная стойкость, лавинный эффект).

## Общий алгоритм

- Старт с аффинных S-box над GF(2^8) (в т.ч. AES‑подобный).
- Эволюционным алгоритмом мутируем перестановки.
- Отбираем кандидатов по метрикам качества и разнообразия профилей.
- В итоге получаем не один лучший S-box, а набор разных сильных» вариантов.

## Метрики

Для каждого S-box считаются:

- **Nonlinearity (NL)** — расстояние до ближайших линейных/аффинных функций.
- **Differential uniformity (DU)** — максимум частот в DDT (меньше — лучше, 4 ≈ уровень AES).
- **Linear bias** — максимум по Уолшу, характеризует линейный криптоанализ.
- **Avalanche effect (AV)** — средняя доля изменившихся выходных бит при флипе одного входного бита (идеал ≈ 0.5).
- **Score** — агрегированная функция качества по всем метрикам.

## Алгоритм

- Генерация стартовой популяции аффинных S-box.
- Вычисление метрик + кэширование.
- Эволюция:
  - выбор элиты с учётом разнообразия;
  - несколько типов мутаций над перестановками;
  - периодические рестарты при стагнации.
- Отдельное хранилище «diverse» кандидатов с контролем минимальной дистанции по метрикам.

## Выходные файлы

- `diverse_sboxes_result.txt` — итоговый список лучших S-box с метриками и выводом в 1D и 16×16 виде.
- `score_dynamics.png` — динамика score.
- `nl_du_dynamics.png` — NL и DU по поколениям.
- `bias_av_dynamics.png` — bias и AV во времени.
- `storage_stagnation_dynamics.png` — заполнение хранилища и стагнация.

## Запуск

Требуется Python 3.10+ и `matplotlib`:

```bash
pip install matplotlib
python your_script_name.py
```

Основные настраиваемые параметры: размер популяции, число поколений, размер элиты, размер хранилища и порог метрики разнообразия.
